# Tealstreet Homebrew Tap

Homebrew tap for [Tealstreet CLI](https://github.com/Tealstreet/cli).

## Installation

```bash
brew tap tealstreet/tap
brew install tealstreet
```

## Updating

```bash
brew update
brew upgrade tealstreet
```

## Uninstall

```bash
brew uninstall tealstreet
brew untap tealstreet/tap
```
